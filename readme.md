## 前言：

### 鼓励自学成才

根据宪法第十九条“鼓励自学成才”的规定，制定本条例(**高等教育自学考试暂行条例**)。

### 任务

高等教育自学考试的任务是通过国家考试促进广泛的**个人自学**和**社会助学**活动，推进在职专业教育和大学后继续教育，造就和选拔德才兼备的专门人才，**提高全民的思想道德和科学文化素质，适应现代化建设的需要**。

## 8.预计的学习安排



> 2020年，虽然我还没有报名成功，但是学习计划也可以列举一下了。
>
> 11门必考课【10门笔试，1门实践考核(毕业论文)】，也就是考9门
>
> 4门选考课【选考英语(二)，直接修满14学分】，也就是考1门
>
> 每年4月一次的《学位英语》1门，共12门笔试。
>
> 
>
> 争取1~2年内考完，也就是1~2个月一本书
>
> 一个月4周，每周学习时间不少于22小时。预计一本书88~176个小时
>
> 一        二        三        四        五        六        日
>
> 2h       2h	    2h       2h       2h       2h        6h
>
> 
>
> 一周22个小时
>
> (风雨无阻，并且每天要坚持锻炼，调整心态，养好身体，不要生病，不要影响日常生活和工作。)
>
> 
>
> 保证休息和睡眠，晚上最晚12点要睡觉，早上7点要起床(第二天还要上班)。
>
> 
>
> 不要脱产自考，如果有工作，工作还是第一位的。
>
> 学生党自便。
